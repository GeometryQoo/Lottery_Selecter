## 🎲 選號機制一：智慧選號器

### 說明
從歷史資料中 **Top 30 最常出現的號碼** 中隨機選取 6 個號碼。

### 操作步驟
1. 點擊「🎲 智慧選號」按鈕
2. 系統自動產生 6 個號碼
3. 顯示紅色號碼球
4. 查看詳細統計表格

### 候選池
- Top 30 最熱門號碼
- 出現機率約在 12-14% 之間

### 特色
- ✅ 基於歷史高頻號碼
- ✅ 操作簡單直觀
- ✅ 顯示詳細統計資訊

---

## 🎯 選號機制二：混合策略選號器

### 說明
結合三種不同策略，提供更平衡的號碼組合。

### 選號規則

| 類型 | 數量 | 顏色 | 來源 | 說明 |
|------|------|------|------|------|
| 🔴 **熱號** | 3 個 | 紅色 | Top 15 號碼 | 歷史高頻號碼 |
| 🔵 **冷號** | 2 個 | 藍色 | 低頻號碼 | 機率 < 11.5% |
| 🟡 **驚喜號** | 1 個 | 金色 | 全部 1-49 | 完全隨機 |

### 操作步驟
1. 點擊「🎯 混合策略選號」按鈕
2. 系統產生 6 個號碼（依上述規則）
3. 顯示三種顏色的號碼球
4. 查看分類統計表格

### 特色
- ✅ 多樣化號碼分布
- ✅ 視覺化分類（三色球）
- ✅ 平衡的選號策略
- ✅ 包含驚喜元素

## 📊 統計資訊說明

### 表格欄位

- **號碼**：選中的號碼（兩位數格式）
- **類型**：
  - 智慧選號：顯示「智慧選號」
  - 混合策略：顯示「🔴 熱號」、「🔵 冷號」或「🟡 驚喜號」
- **出現次數**：該號碼在歷史 2,056 期中出現的次數
- **機率(%)**：出現次數除以總期數的百分比
- **排名**：在 49 個號碼中的出現頻率排名

### 統計摘要
- **平均出現機率**：選中 6 個號碼的平均出現機率
- **策略說明**：本次選號使用的策略
- **重要提醒**：中獎機率說明

## ⚠️ 重要提醒

### 關於中獎機率

**無論使用哪種選號策略，中獎機率都是固定的：**

```
大樂透頭獎中獎機率 = 1 / C(49, 6) = 1 / 13,983,816

≈ 0.0000072%
```

### 統計數據的意義

- ✅ 歷史統計僅供參考
- ✅ 每次開獎都是獨立隨機事件
- ✅ 過去的頻率不代表未來趨勢
- ✅ 所有號碼組合中獎機率相同

### 理性購彩

- 🎲 當作娛樂，量力而為
- 📊 統計分析是學習工具
- 🚫 不要過度投入
- ✨ 享受選號的樂趣

## ❓ 常見問題

### Q1: 為什麼雙擊 .command 檔案沒有反應？
**A**: 請檢查：
1. 是否已設定為可執行（`chmod +x 啟動工具.command`）
2. 系統安全性設定是否允許執行
3. lottery.db 是否在同一目錄

### Q2: 出現「找不到 Python 3」錯誤？
**A**: 請先安裝 Python 3：
```bash
# macOS 使用 Homebrew
brew install python3
```

### Q3: 可以重複選號嗎？
**A**: 可以！點擊按鈕即可產生新的號碼組合。

### Q4: 如何關閉程式？
**A**: 點擊視窗的關閉按鈕（X）即可。

### Q5: 這些號碼真的比較容易中獎嗎？
**A**: 不會！所有號碼組合的中獎機率都相同。本系統僅提供基於歷史統計的選號參考，增加選號樂趣。
